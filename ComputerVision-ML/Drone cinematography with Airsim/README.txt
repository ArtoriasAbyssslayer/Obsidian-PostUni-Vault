Thank you for participating in the CVML Workshop on Deep Learning and Computer Vision 2022!

In order for you to be able to actively participate in the "Drone cinematography with Airsim" workshop, you may prepare a Windows PC (Linux is not recommended, but can be managed) with the following requirements:
- A capable NVIDIA GPU (at least 4 GiB of memory)
- AirSim simulator with related software (Unreal Engine 4.27, VisualStudio 2022, etc) installed following the official installation documentation: https://microsoft.github.io/AirSim/build_windows/
- The Blocks Environment tested following the "How to Use AirSim" section featured on https://microsoft.github.io/AirSim/build_windows/
- Anaconda installed (preferably miniconda, follow the installation instructions: https://docs.conda.io/en/latest/miniconda.html)
- Two anaconda evironments with python 3.8 created (see https://docs.conda.io/projects/conda/en/4.6.0/_downloads/52a95608c49671267e40c689e0bc00ca/conda-cheatsheet.pdf):
	i) One named 'airsim' in which the following packages must be installed: numpy, opencv, pillow, msgpack-rpc-python.
	ii) One named 'segmentation' in which the following packages must be installed: jupyter, opencv, pillow, tensorflow. You must be able to activate this conda environment from inside a jupyter notebook.
- All the files and folders downloaded from the current google drive directory.
